# 🚀 NexaAI Studio Website

A modern, responsive, and visually stunning AI-focused business website built using HTML, CSS, Bootstrap, JavaScript, and jQuery. NexaAI Studio showcases innovative AI-powered creative solutions, including AI-generated branding, logo evolution, visual storytelling, and digital marketing services. The website features smooth animations, interactive sections, modern UI components, and a fully responsive design optimized for all devices.

## 🌐 Live Demo

🔗 https://rukshana-roshan.github.io/nexa-Ai-studio-website/
